Данный скрипт представляет собой консольную реализацию телефонного справочника со следующими возможностями:

 - Постраничный вывод записей из телефонного справочника (по 10 записей на странице)
 - Возможность добавления новой записи в телефонный справочник
 - Возможность редактирования записи в телефонном справочнике
 - Возможность удаления записи(ей) из телефонного справочника
 - Возможность поиска записи(ей) в телефонном справочнике

Особенности реализации:

1) Записи телефонного справочника хранятся в файле Phonebook.txt (кодирование 'UTF-8') со следующим форматом:
фамилия, имя, отчество, название организации, телефон рабочий, телефон личный (сотовый)
2) Записи телефонного справочника регистрочувствительны!
3) Уникальным критерием записей телефонного справочника является атрибут 'телефон личный(сотовый)'.
